# Overview

The `SpFut.covariates` package contains thirteen functions to easily download and process climate, anthropological, and environmental covariates data.

- `get_buffered_bbox`
- `cleaning`
- `get_climate`
- `get_elevation`
- `get_flowlines`
- `get_footprint`
- `get_landcover`
- `get_population`
- `get_protectedAreas`
- `get_roads`
- `get_soil`
- `get_traveltime`
- `get_waterbodies`

The first two functions are used for data setup and cleaning. The remaining functions are wrappers to access, download, and format covariate data. 

See the vignette for a demonstration on how to use the functions in the `SpFut.covariates` package.
